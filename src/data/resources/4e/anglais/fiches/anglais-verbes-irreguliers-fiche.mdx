# 🇬🇧 Les verbes irréguliers anglais

En anglais, beaucoup de verbes courants sont **irréguliers** : leur prétérit et leur participe passé ne suivent pas la règle du **-ed** des verbes réguliers.

**Il faut les apprendre par cœur !**

<Glossaire>
  <GlossaireItem term="base form / infinitive">forme de base = l'infinitif (ex : *go*)</GlossaireItem>
  <GlossaireItem term="preterite (simple past)">prétérit = passé simple (ex : *went*)</GlossaireItem>
  <GlossaireItem term="past participle">participe passé (ex : *gone*)</GlossaireItem>
  <GlossaireItem term="irregular verb">verbe irrégulier — ne prend pas -ed au prétérit</GlossaireItem>
  <GlossaireItem term="regular verb">verbe régulier — prend -ed au prétérit (ex : *play → played*)</GlossaireItem>
</Glossaire>

---

<Section title="Les 3 formes à connaître" color="blue" emoji="📌" index={1}>

Pour chaque verbe irrégulier, il faut retenir **3 formes** :

<Formule>
infinitif → prétérit → participe passé
</Formule>

<Exemple>
*go → went → gone* (aller)

*eat → ate → eaten* (manger)

*cut → cut → cut* (couper)
</Exemple>

<Astuce>
Le participe passé s'utilise avec **have/has** (present perfect) ou **be** (passif) :
- *I have **eaten** already.* → J'ai déjà mangé.
- *The cake was **eaten**.* → Le gâteau a été mangé.
</Astuce>

</Section>

<Section title="Groupe 1 — Verbes invariables" color="green" emoji="🟰" index={3}>

Ces verbes ont **la même forme** pour les 3 colonnes.

<Retenir title="Astuce mémo">
**cut – cut – cut** → Si tu connais l'infinitif, tu connais tout !
</Retenir>

<Tableau>

| 🇫🇷 Français | Infinitif | Prétérit | Participe passé |
|---|---|---|---|

| laisser | **let** | let | let |

</Erreur>

<Section title="Récapitulatif des astuces" color="amber" emoji="💡" index={6}>

<Astuce title="Groupe 1 — invariable">
L'infinitif = le prétérit = le participe → **apprends juste l'infinitif !**
</Astuce>

<Astuce title="Groupe 2 — 2 formes identiques">
Le prétérit = le participe passé → **apprends l'infinitif + une seule autre forme.**
</Astuce>

<Erreur title="Groupe 3 — toutes différentes">
Ces verbes ont 3 formes distinctes → **apprends les 3 colonnes par cœur.**
</Erreur>

<Astuce title="Méthode recommandée">
Répète chaque ligne à voix haute :

→ *go — went — gone — aller*

→ *eat — ate — eaten — manger*
</Astuce>

</Section>
