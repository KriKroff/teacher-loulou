# 🇪🇸 Le futur de l'indicatif (el futuro de indicativo)

Le futur permet d'exprimer : **une action à venir, une promesse ou une prédiction**.

<Glossaire>
  <GlossaireItem term="el futuro">le futur</GlossaireItem>
  <GlossaireItem term="el infinitivo">l'infinitif</GlossaireItem>
  <GlossaireItem term="la terminación">la terminaison</GlossaireItem>
  <GlossaireItem term="regular">régulier</GlossaireItem>
  <GlossaireItem term="irregular">irrégulier</GlossaireItem>
</Glossaire>

---

<Section title="Quand utiliser le futur ?" color="blue" emoji="🕐" index={1}>

On utilise le futur pour :

- **Une action qui se passera dans l'avenir** : *Mañana iré al colegio.* (Demain j'irai au collège.)
- **Une promesse** : *Te llamaré esta noche.* (Je t'appellerai ce soir.)
- **Une prédiction** : *Creo que lloverá.* (Je crois qu'il pleuvra.)

<Astuce>
En français : "j'irai" = en espagnol : "iré". Le lien est direct !
</Astuce>

</Section>

---

<Section title="Le futur simple — verbes réguliers" color="green" emoji="📐" index={2}>

**La règle d'or** : pour TOUS les verbes réguliers (-AR, -ER, -IR), on garde l'**infinitif entier** et on ajoute les terminaisons.

<Formule>
infinitif + **é / ás / á / emos / éis / án**
</Formule>

<Retenir>
Toutes les personnes portent un **accent écrit** sauf **nosotros** (-emos).
</Retenir>

<Tableau>

| Pronom | hablar | comer | vivir |
|---|---|---|---|
| yo | hablar**é** | comer**é** | vivir**é** |
| tú | hablar**ás** | comer**ás** | vivir**ás** |
| él/ella | hablar**á** | comer**á** | vivir**á** |
| nosotros | hablar**emos** | comer**emos** | vivir**emos** |
| vosotros | hablar**éis** | comer**éis** | vivir**éis** |
| ellos/ellas | hablar**án** | comer**án** | vivir**án** |

</Tableau>

<Exemple>
*Mañana **hablaré** contigo.* → Demain je te parlerai.

*Ellos **comerán** en casa.* → Ils mangeront à la maison.

*Nosotros **viviremos** en Madrid.* → Nous vivrons à Madrid.
</Exemple>

</Section>

<Section title="Le futur simple — verbes irréguliers" color="orange" emoji="⚡" index={3}>

Les verbes irréguliers ont un **radical modifié**, mais gardent exactement les **mêmes terminaisons** que les réguliers !

<Astuce>
Mémorise le radical irrégulier, les terminaisons ne changent pas.
</Astuce>

<Tableau>

| Infinitif | Radical futur | Exemple (yo) |
|---|---|---|
| caber | cabr- | cabr**é** |
| decir | dir- | dir**é** |
| haber | habr- | habr**é** |
| hacer | har- | har**é** |
| poder | podr- | podr**é** |
| poner | pondr- | pondr**é** |
| querer | querr- | querr**é** |
| saber | sabr- | sabr**é** |
| salir | saldr- | saldr**é** |
| tener | tendr- | tendr**é** |
| valer | valdr- | valdr**é** |
| venir | vendr- | vendr**é** |

</Tableau>

<Exemple>
*¿**Podrás** venir mañana?* → Pourras-tu venir demain ?

***Tendré** que estudiar.* → Je devrai étudier.

***Haremos** los deberes juntos.* → Nous ferons les devoirs ensemble.
</Exemple>

<Erreur>
Ne jamais écrire *\*haceré* → la forme correcte est **haré** (radical **har-**).

Ne jamais écrire *\*poderán* → la forme correcte est **podrán** (radical **podr-**).
</Erreur>

</Section>

<Section title="Le futur proche" color="purple" emoji="🚀" index={4}>

Le futur proche exprime une action **imminente ou planifiée**.

**Structure** : **IR (présent) + a + infinitif**

<Formule>
voy / vas / va / vamos / vais / van + **a** + verbe à l'infinitif
</Formule>

<Tableau>

| Pronom | IR (présent) | Futur proche |
|---|---|---|
| yo | voy | voy **a** estudiar |
| tú | vas | vas **a** comer |
| él/ella | va | va **a** salir |
| nosotros | vamos | vamos **a** hablar |
| vosotros | vais | vais **a** vivir |
| ellos/ellas | van | van **a** leer |

</Tableau>

<Exemple>
*Los alumnos **van a callarse**.* → Les élèves vont se taire.

***Voy a estudiar** esta noche.* → Je vais étudier ce soir.

*¿**Vas a venir** a la fiesta?* → Tu vas venir à la fête ?
</Exemple>

<Astuce>
C'est exactement comme le français **"aller + infinitif"** :
"Je vais partir" = "Voy a salir". Très facile !
</Astuce>

</Section>

---

<Section title="Futur simple vs Futur proche" color="cyan" emoji="⚖️" index={5}>

Les deux formes existent en espagnol, comme en français.

<Tableau>

| | Futur simple | Futur proche |
|---|---|---|
| **Structure** | infinitif + terminaison | IR + a + infinitif |
| **Exemple** | *Estudiaré mañana.* | *Voy a estudiar mañana.* |
| **Traduction** | J'étudierai demain. | Je vais étudier demain. |
| **Usage** | action future, promesse | action planifiée, imminente |

</Tableau>

<Retenir>
Les deux formes sont souvent **interchangeables** en espagnol. Le futur proche (IR + a + inf) est très courant à l'oral.
</Retenir>

</Section>
